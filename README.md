## 📋 نظرة عامة
تطبيق للتعلم الآلي يتوقع أداء الطالب (الدرجات) بناءً على عوامل مختلفة.

## 🎯 المدخلات
- **Gender**: Male/Female
- **Parental Education Level**: Bachelor/Master/High School
- **Lunch Type**: Standard/Free
- **Test Preparation Course**: Completed/None
- **Study Time**: وقت الدراسة بالدقائق
- **Absences**: عدد أيام الغياب

## 📊 المخرجات
- Math Score
- Reading Score
- Writing Score
